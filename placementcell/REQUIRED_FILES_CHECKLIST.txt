================================================================================
COMMON ERRORS CAUSED BY MISSING FILES
================================================================================

Error: "Undefined variable $pg_courses_grouped"
File Missing: course_groups.php
Fix: Get this file from the sender

Error: "Call to undefined function"
File Missing: Likely missing an include file (config.php, header.php, etc.)
Fix: Verify all files were copied

Error: "Table doesn't exist"
File Missing: SQL scripts not run
Fix: Run the SQL scripts in phpMyAdmin

Error: "Cannot modify header information"
File Missing: None - different issue (usually spacing/BOM in PHP files)
Fix: Check for extra spaces before <?php tags
